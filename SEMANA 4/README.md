## Descripción del Proyecto

El presente proyecto consiste en el desarrollo de una página web moderna, estilizada y completamente responsiva utilizando HTML5, CSS3 avanzado y Bootstrap 5.  
La página cumple con todos los requisitos establecidos en la actividad correspondiente a la Semana 04, incluyendo el uso del sistema de grid, componentes interactivos, animaciones CSS y su publicación mediante GitHub Pages.

---

## Objetivos de la Actividad

- Aplicar técnicas avanzadas de CSS3 como el modelo de cajas, animaciones, transiciones y media queries.
- Dominar el sistema de Grid y Flexbox de Bootstrap para lograr un diseño fluido y adaptable.
- Utilizar componentes interactivos de Bootstrap como navbar, buttons, cards, collapse y form.
- Personalizar estilos de Bootstrap mediante CSS externo.
- Implementar y publicar el proyecto en un repositorio público de GitHub con acceso desde GitHub Pages.
