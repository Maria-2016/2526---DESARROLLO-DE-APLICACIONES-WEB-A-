# Galería Interactiva con JavaScript

Proyecto desarrollado para la Semana 05: Fundamentos de JavaScript – Manipulación del DOM y Eventos.

## Funcionalidades
- Agregar imágenes mediante URL.
- Seleccionar una imagen (solo una a la vez).
- Eliminar la imagen seleccionada.
- Atajo de teclado: tecla Delete.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
